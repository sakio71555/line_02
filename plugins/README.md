# Plugin Distribution Notes

This repository does not currently distribute Codex plugins.

Current state:

- Not distributed.
- No manifest.
- No install script.
- No npm package.
- No marketplace settings.
- No automatic setup script.

Future use:

- Consider plugin packaging only if the team needs to share Codex operating rules across repositories.
- Keep product runtime, API, UI, DB, and external service behavior separate from any future Codex plugin packaging.
