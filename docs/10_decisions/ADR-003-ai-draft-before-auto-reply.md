# ADR-003: AI Draft Before Auto Reply

## Status

Accepted

## Context

住宅相談では、見積金額、土地価格、建売在庫、補助金、保証、契約条件など、AIが断定すると危険な話題が多く含まれます。

## Decision

初期AI機能は自動返信ではなく、会話要約、返信下書き、次アクション提案を中心にします。自動返信は `response_mode = bot_auto` の場合だけ許可し、人間対応中は送信しません。

## Consequences

- 担当者支援として導入しやすい。
- 誤回答リスクを下げられる。
- 自動化の効果は段階的になる。
- 自動返信を拡張する前に、AI下書きの品質レビューが必要になる。
